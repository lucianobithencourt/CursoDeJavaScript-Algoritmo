Eventos com mouse - DOM

Evento - onclick

onclick - Produz-se quando se clica o botão do mouse sobre um elemento da página, geralmente um botão ou um link. Javascript 1.0

Evento - onmousedown

onmousedown - Produz-se o evento onmousedown quando o usuário clica sobre um elemento da página. onmousedown se produz no momento de clicar o botão, soltando ou não. Javascript 1.2

Evento - onmousemove

onmousemove - Produz-se o evento quando o mouse se move pela página. Javascript 1.2 Javascript 1.2

Evento - onselect

onselect - Executa-se quando um usuário realiza uma seleção de um elemento de um formulário. Javascript 1.0 Javascript 1.2
